# Dogger App

## Install

### Frontend

#### Install node dependencies
```bash
cd path/Dogger/frontend
npm install
```

#### Run project
```bash
npm run start
```

#### Build ***Dogger*** to production
```bash
npm run build
```

### Backend

Create virtualenv
```bash
virtualenv dogger
```
Install dependencies of __Django__
```bash
(dogger) cd path/project/backend
(dogger) pip install -r requirements.txt
```
